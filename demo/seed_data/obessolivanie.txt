Методы обессоливания воды для обогатительных фабрик

Иванов А.В., Петров С.Н. (2022) "Технологии водоподготовки на горно-обогатительных предприятиях", Журнал горной науки, том 18, №4, стр. 45-58.

Аннотация: Рассмотрены основные методы обессоливания шахтных и карьерных вод для обогатительных фабрик. Сравниваются обратный осмос, нанофильтрация, ионный обмен и электродиализ.

1. Обратный осмос (RO)
Применяется для удаления растворённых солей с эффективностью 95-99%. Рабочее давление 10-60 бар. Подходит для исходной воды с общим солесодержанием до 3000 мг/л. Сухой остаток на выходе ≤ 50 мг/дм³. Капитальные затраты: 50-80 млн руб. для установки производительностью 100 м³/сут. Необходима предварительная очистка от взвешенных веществ и свободного хлора.

2. Нанофильтрация (NF)
Эффективна для удаления двухвалентных ионов (Ca²⁺, Mg²⁺, SO₄²⁻) на 90-98%, одновалентных (Na⁺, Cl⁻) на 30-60%. Рабочее давление 5-25 бар. Подходит для воды с сульфатами 200-500 мг/л. Сухой остаток на выходе 200-500 мг/дм³. Применима при требуемом сухом остатке ≤ 1000 мг/дм³.

3. Ионный обмен
Использует ионообменные смолы (катиониты КУ-2-8, аниониты АВ-17-8). Подходит для финишной очистки после RO/NF. Регенерация раствором NaCl или HCl. Ограничение: высокий расход реагентов при солесодержании > 1000 мг/л.

4. Электродиализ
Эффективен для вод с солесодержанием 500-5000 мг/л. Расход электроэнергии 0.5-3 кВт·ч/м³. Применяется для частичного обессоливания и снижения сульфатов до 300 мг/л.

Вывод: для исходной воды с сульфатами 200-300 мг/л, хлоридами 200-300 мг/л, Ca/Mg/Na по 200-300 мг/л и требуемым сухим остатком ≤ 1000 мг/дм³ оптимальным является метод нанофильтрации с возможным сочетанием с ионным обменом на финишной стадии.
