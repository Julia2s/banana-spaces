# Shared contracts

Контракты в этой директории имеют статус draft до командной синхронизации Sync 1.

Цель текущего слоя — зафиксировать минимальные границы MVP-потока: ingestion task, normalized artifact, нормализованный документ, source span, retrieval document, Query IR, evidence bundle, retrieval run и answer payload. До заморозки контрактов изменения должны проходить через явное решение команды.
